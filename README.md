# eng-skills

A collection of Claude Code skills focused on software engineering quality — code review, refactoring, architecture design, and design patterns.

## Skills

| Skill | Description |
|-------|-------------|
| `unified-code-review-pro` | PR/diff review coordinator. Auto-selects review depth and runs parallel checks across correctness, security, architecture, and efficiency. |
| `deep-module-refactor-advisor` | Audits mature codebases for shallow modules and high coupling. Produces facade encapsulation recommendations without touching code. |
| `module-structure-optimizer` | Post-feature optimization: finds encapsulation candidates, removes redundant abstractions, detects N+1 and sequential-await risks. |
| `design-patterns` | Context-aware design pattern selector. Recommends AI-friendly patterns and steers away from over-engineering. |
| `ai-native-design` | Language-agnostic AI-native architecture principles for module boundaries, layered responsibilities, stable interfaces, and test strategy. |
